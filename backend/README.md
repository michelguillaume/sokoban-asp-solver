## A* solver (Level 36)

Ce dossier contient aussi un petit script **A\*** (pur Python) qui résout le **Level 36** (official levels) en cherchant dans l’espace des **pushes**.

### Exécuter

Depuis `new-backend/`:

```bash
python astar_level36.py
```

Le script affiche la solution sous forme d’une **suite de directions** `U/D/L/R` (chaque lettre = 1 push), plus quelques stats.
